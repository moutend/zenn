## はじめに

本記事では視覚に障害のある方を対象としてM5Stack CoreS3のハードウェア構成とファームウェア書き込みの手順を説明します。ボタンやスイッチの配置については手で触って確認いただけるように説明しますので、ご参考になれば幸いです。  

## M5Stackの種類とCoreS3をおすすめする理由

M5StackはIoT開発向けの小型コンピューターです。いくつか種類があり、本記事で扱うCoreS3は筐体にタッチパネルやボタン、各種センサーが搭載された完成品として販売されています。  
CoreS3の寸法は縦横が54 mm、厚みが31.5 mm（後述のDIN Baseバッテリーユニットを除外すると16 mm）です。AppleのAirPodsイヤホンの充電ケースをひとまわり大きくした程度のサイズ感です。  
他のモデルには基板がむき出しの産業用モデルなどがあります。全盲や重度の弱視者の場合、自力ではんだ付けや組み立てを行うのは困難かと思いますので、プロトタイプの作成が目的でしたらCoreS3をおすすめします。  

### （補足）CoreS3の種類

記事を投稿した時点でCoreS3には以下の種類があります。それぞれスイッチサイエンスの通販サイトにリンクしています。  

1. [CoreS3本体 + DIN Base（500 mAhバッテリー内蔵）](https://www.switch-science.com/products/8960)
2. [CoreS3 SE本体のみ](https://www.switch-science.com/products/9690)
3. [CoreS3用バッテリーボトム単体](https://www.switch-science.com/products/9421)
4. [CoreS3 Lite](https://www.switch-science.com/products/10610)

筆者のおすすめは1番目のモデルです。CoreS3はバッテリーを着脱できる構造になっています。1番目のモデルはバッテリーが装着済みの状態で販売されていますので、なにかと便利に使えます。  
なお、バッテリーが必要なくても1番目のモデルをお勧めします。CoreS3SEは廉価版モデルです。バッテリーが装着されていない他に、各種センサーが本体に搭載されていません。  
余談になりますが、M5Stack公式のディストリビュータ一覧で日本の販売店としてスイッチサイエンスが掲載されています。価格最優先なら海外直販や他の通販が安い場合もありますが、正規流通、国内配送、日本語サポートの点では無難な選択かと思われます。筆者もスイッチサイエンスでCoreS3を購入しました。  

## 電源の入れ方

M5Stack CoreS3が届いたら箱から取り出してください。ツルツルした平らな面がタッチパネル（タッチ操作に対応した画面）です。  
続いて、本体側面を指で触って細長い窪みを探します。それがUSB type Cポートです。ポートが見つかったら、CoreS3と充電器、またはPCをUSBケーブルで接続します。  
USBケーブルを接続した時点でCoreS3の電源が入り、工場出荷時のデモ用プログラムが起動します。通常はケーブルを接続すると即座に起動します。電源ボタンを押すといった操作は必要ありません。  
起動は数秒で完了します。適当にタッチパネルを触ってみてください。CoreS3本体から効果音が再生されますので、それで正常に起動できたと判断できます。  

### CoreS3本体の側面レイアウト

CoreS3本体の側面に配置されているボタンなどについて説明します。タッチパネルが前面、USB type Cポートが上にある状態での配置は次のとおりです。  

- 上側面: 拡張ポート、USB type Cポート、POWERボタンが配置されています。
- 左側面: micro SDカードスロット、RSTボタンが配置されています。
- 下側面: スピーカー開口部が配置されています。小さな穴が多数あり、ザラザラとした手触りです。
- 右側面: 何もありません。

CoreS3に装着されているDIN Base（バッテリーユニット）の側面は次のとおりです。

- 上側面: 何もありません。
- 左側面:
  - DC電源ジャックとバッテリーのON / OFFスライドスイッチが配置されています。
  - USB type Cポートがある方向にスライドするとOFF、反対側にスライドするとONです。
  - DIN BaseバッテリーがONの状態で本体のPOWERスイッチを押すと電源が入りプログラムが起動します。動作を停止させるにはスライドスイッチをOFFにしてください。
- 下側面: 何もありません。
- 右側面: 拡張ポートが2つ配置されています。

## ファームウェア書き込みモードでPCに接続する

電源が入り、CoreS3に初期不良がないことを確認したら、次はファームウェア書き込みモードでの接続が可能か確認します。

1. PCとCoreS3をUSBケーブルで接続します。工場出荷時のデモ用プログラムが起動しますが、無視して構いません。
2. 本体側面のRSTスイッチを3秒間押し続けます。
3. CoreS3がUSBモデムとして認識されるようになり、ファームウェアの書き込みが可能な状態になります。
  - macOSの場合: Terminal.appを開いて`ls /dev/cu.*`を実行します。結果に`/dev/cu.usbmodem2101`が表示されたら成功です。
